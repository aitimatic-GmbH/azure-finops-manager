# Contributing

## No External Contributions

This repository **does not accept external contributions**.

It serves as a reference implementation and a portfolio project for aitimatic GmbH.

## What You Can Do

You are welcome to:

* ✅ **Fork this repository** and create your own version
* ✅ **Use the code** under the terms of the MIT License
* ✅ **Open issues** for questions (note: no guarantee of response)

## What We Don't Accept

* ❌ Pull requests
* ❌ Feature requests
* ❌ Bug fixes from external contributors

## Purpose

This repository showcases our internal approaches and solutions. We maintain full control over the codebase to ensure **quality, consistency, and alignment with internal standards**.

## Need Help?

If you want to use this code for your own implementation:

1. Fork the repository
2. Review the documentation
3. Adapt the code to your needs

**Use at your own risk. No support provided.**
